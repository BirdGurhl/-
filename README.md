# 记账本
## 技术栈
```
node.js、express + express-gennerator、HTML + EJS模板引擎、Mongondb + Mongoose、apipost接口测试工具
```
## 部署及运行
```
1、npm i
```
```
2、创建数据库（配置在config.js）,默认数据库名demo ，运行脚本文件 sql.js导入数据（可选）
```
```
3、运行 npm start   访问地址：http://127.0.0.1:3000/
```
